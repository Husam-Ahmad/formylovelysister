# Auto Call Button Clicker Extension

This Chrome/Edge extension automatically detects and clicks the "Call" button when a patient appears on the page.

## How to install and use

1. Open Chrome or Edge and go to `chrome://extensions/` or `edge://extensions/`.
2. Enable "Developer mode" (top right).
3. Click "Load unpacked" and select the `auto-call-extension` folder.
4. Visit the target web page. The extension will automatically click the "Call" button when it appears.

## Customization
- If the button uses a different label or selector, edit `content.js` and adjust the detection logic.
